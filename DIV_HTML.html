Use external style to apply the global formatting of embedded style to multiple pages. External style allows you to specify formats and apply them to multiple Web pages rather than just one. External style is also know as linked style, because instead of listing style specification at the top of your Web page, you create a link to an external document that contains the style code, known as external stylesheet. This method allows you to format a set of Web pages, such as a Web publication, with a uniform style and allows you to change the syle for all pages later simplt by editing the external style sheet. External style can be used in a page togerther with both inline and embedded styles.
